## Getting Started
### Prerequisites
- **Python 3.9+** (added to PATH)
- **Node.js & npm** (for the React frontend)
- **Git** (optional, for version control)

### 1пёЏвѓЈ Clone the Repository
```bash
git clone <repository_url>
cd EduMind
```

### 2пёЏвѓЈ Set Up the Backend
```bash
# Create and activate a virtual environment
python -m venv venv
# Windows
.\\venv\\Scripts\\activate
# macOS/Linux
source venv/bin/activate

# Install Python dependencies
pip install -r requirements.txt
```

### 3пёЏвѓЈ Configure Environment Variables
Create a `.env` file in the project root:
```dotenv
SECRET_KEY=your-secret-key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
```

### 4пёЏвѓЈ Seed the Demo Database
```bash
python seed_demo.py
```
The script creates users, categories, sample documents, quizzes, comments, notifications, and a pending document for moderation.

### 5пёЏвѓЈ Run the Backend Server
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```
Backend API will be available at **http://localhost:8000**.

### 6пёЏвѓЈ Run the Frontend
```bash
cd frontend
npm install
npm run dev   # Vite dev server on http://localhost:5173
```
Open the URL in your browser to explore the portal.
